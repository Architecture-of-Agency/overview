# PhD Urban Governance Research System

Complete system for Web3-enabled urban governance research.

## Quick Start

### Obsidian
- Open `obsidian-vault/` in Obsidian app
- Start taking notes!

### Website
```bash
cd website
npm install
npm run dev
```

### Python
```bash
cd scripts
pip install -r requirements.txt
python data-processing/data_processor.py
```

### Deploy Website
- Push to GitHub
- Connect to Vercel
- Done!

## Structure
- `obsidian-vault/` - Research notes
- `website/` - Public research site
- `scripts/` - Data processing
- `contracts/` - Smart contracts

Built for iPad Pro + GitHub Codespaces workflow.
